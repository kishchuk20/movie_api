API Endpoint Documentation

Request URL Description
GET All Movies /movies Returns a list of all movies.
GET a Movie By a Title /movies/:title Returns a movie and details about it by a title.
GET Movies by Type /movies/type/:typeName Returns movies by type.
POST a New User /users Creates a profile for a new user.
GET All Users /users Returns a list of all users.
PUT New Info /users/:username Updates user's info.
POST Favorite Movies /users/:username/movies/:movieID Adds a movie to a user's favorites.
DELETE Favorites /users/:username/movies/:movieID Deletes a movie from user's favorites.
DELETE an Account /users/:username Lets user to delete an account.
